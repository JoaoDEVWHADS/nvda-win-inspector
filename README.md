# NVDA Window Inspector

> **Repository**: [nvda-win-inspector]("/JoaoDEVWHADS/nvda-win-inspector")
> **Author**: JoaoDEVWHADS
> **Version**: 2026-01-14

## Description
Addon to inspect window name and focus object class via `NVDA+Control+Shift+D`.

## Setup
Run the setup script to initialize your environment:
```bash
./setup.sh
source .venv/bin/activate
```

## Usage
Press `NVDA+Control+Shift+D` to hear the current foreground window name and the focused object's window class.
